为什么选择 Hexo + GitHub Pages 搭建Blog

轻量：没有麻烦的配置，使用标记语 Markdown ，无需自己搭建服务器
免费：免费托管 Github 仓库
通用：是目前比较流行的方案，社区活跃，不断创新

一、Hexo、GitHub Pages 介绍

Hexo 一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

GitHub Pages 一个静态网站托管服务，用来管理你的一个GitHub 库的个人的、组织的、或者项目的页面（免费），当然白嫖肯定是有限制的。

Github Pages 限制：
发布的网站不能超过1 GB。
网站每月100GB的带宽限制。
每小时小于10个版本。
